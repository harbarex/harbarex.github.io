NarcoSoft — Drug-Abuse Prevention

An Android platform built with the Government of Punjab to combat drug abuse.

Nov 2017 Software

NarcoSoft was built in collaboration with the Special Task Force of the Government of Punjab for the Drug Abuse Prevention Officer (DAPO) program.

  • Served as an information-retrieval platform with role-based access for volunteers, police officers and civilians.
  • Visualised the current status of the known population for each locality (Mohalla).
  • Developed in Java with data stored on a remote server at the Government of Punjab's IT HQ.
